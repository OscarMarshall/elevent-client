# eleventclient

A Hoplon project designed to...well, that part is up to you.

## Dependencies

- java 1.7+
- [boot][1]
- [leiningen][2]
- [bower][3]

## Usage

1. Download Bower dependencies:

    ```bash
    $ bower install
    ```

2. Start the auto-compiler. In a terminal:

    ```bash
    $ boot development
    ```

3. Go to [http://localhost:8000][4] in your browser.

4. You're already done.
